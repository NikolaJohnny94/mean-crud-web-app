## Description 📜

MEAN CRUD app with Bootstrap and Sass 🤗

## Requirements ⚙️

Install node modules: <br>

```
npm install
```

### Mongo DB URL 🍃

In the ./backend/config/.env file, add your Mongo URL

## Run the dev server 👨‍💻

### Back-End 🌐

Navigate to **backend** folder and run:

```
npm run dev
```

### Front-End 🖼️

Navigate to **frontend** folder and run:

```
npm start
```
